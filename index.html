Type de monture

Sélectionnez le type de monture

Choisir le type de monture
Cerclée
Monture classique avec cercle complet autour du verre. Compatible avec tous les indices et tous les types de verres. Aucune contrainte technique.
Tous indices disponibles · Pas de précalibrage obligatoire
Nylor (semi-percée)
Monture avec fil nylon en bas du verre. Le verre doit être suffisamment résistant pour supporter la tension du fil sans se fissurer.
Indice minimum 1.60 · Précalibrage obligatoire
Percée (sans cercle)
Monture vissée directement dans le verre. Nécessite un verre épais et résistant aux contraintes mécaniques. Le verre est exposé sans protection.
Indice minimum 1.60 · Précalibrage obligatoire
